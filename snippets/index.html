Créer des snippets avec Visual Studio Code

Utilité des snippets

Laïus sur l'utilité de gagner du temps

Où créer un snippet

  • Sur MAC : En haut à gauche Code -> Preferences -> User Snippets -> Dans le champ de saisie choisis un langage
  • Sur windows : En haut à gauche File -> Preferences -> User Snippets -> Dans le champ de saisie choisis un langage

Fonctionnement JSON

Le snippet est délimité par des accolades {, on rentre les données lignes après lignes, chaque ligne se termine par une virgule sauf la dernière (soyez attentif).

Les propriétés et leur(s) valeur(s) sont encadrées par des guillemets doubles et deux point : séparent la propriété de leur(s) valeur(s).

Dans les guillemets les espaces comptent et permettent d'indenter le code, les tabulations ne sont elles pas acceptées.

Il est possible de guider le curseur de saisie afin de remplir le code de manière fluide lors de l'utilisation du snippet, pour ce faire nous utiliseront dollard + n. Comme ceci: $1, $2 etc...

Création de snippet

Evidement demandez aux apprenants de remplir les métas content à leur guise et changez / ajouter ce qui vous semble néccessaire

{
    // Snippets : enregistre du code pour simplifier la saisie ultérieur de code, automatisation des tâches, 
    le code est enregistré dans un fichier JSON {propriété: valeur}
    // Sur MAC : Code -> Preferences -> User Snippets -> choisis un langage
    // Sur windows : File -> Preferences -> User Snippets -> choisis un langage

    // Place your snippets for html here. Each snippet is defined under a snippet name and has a prefix, body and 
    // description. The prefix is what is used to trigger the snippet and the body will be expanded and inserted. Possible variables are:
    // $1, $2 for tab stops, $0 for the final cursor position, and ${1:label}, ${2:another} for placeholders. Placeholders with the 
    // same ids are connected.

    // Exemple:
    "Print to console": {
        "prefix": "docperso", // mot clef à taper pour afficher le code
        "body": [
            "", // retour à la ligne automatique après la virgule
            "",
            "",
            "    ",// sensible à la case
            "    ",
            "    ",
            "    ",// $1 -> $n pour guider le curseur de saisie (tab pour se déplacer)
            "    ",
            "    ",
            "    $6",
            "    $7",
            "    ",
            "",
            "",
            "    $8",
            "    ",
            "",
            ""// virgule non neccessaire à la fin
        ],
        "description": "Custom doctype with 'sonia' as author"// note pour donner des infos sur le code pré-enregisré
    }
}
